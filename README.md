# Student Study Planner

This project is a simple website that helps students organize their study tasks and deadlines in an easy way.

The main idea is to give the student a clear overview of what they need to do, what is coming soon, and what subjects they are currently taking.


## Project Description

The website is designed to be simple and clean. It focuses only on the important things such as tasks, deadlines, and subjects without adding complex features.

The goal is to make it easy for any student to quickly check their study progress and stay organized.


## Pages
The project includes 4 main pages:

- Dashboard: shows a summary of tasks, subjects, and deadlines  
- Subjects Page: allows adding and viewing subjects  
- Tasks & Exams Page: used to manage tasks and exams  
- Deadlines Page: shows upcoming deadlines and progress  

## Features
- Greeting message
- Summary of unfinished tasks  
- List of important tasks  
- Subjects overview  
- Ability to add, delete, and complete tasks  
- Deadlines tracking  


## Technologies Used
- HTML  
- CSS  
- JavaScript
